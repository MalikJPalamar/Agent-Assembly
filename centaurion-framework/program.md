# Centaurion Framework — Autoresearch Program

> Adapted from [karpathy/autoresearch](https://github.com/karpathy/autoresearch) methodology.


## Development Methodology

### Phase 1: Spec-Driven Prototype (Current → Milestone 1)
Daily agent schedule:
1. Morning scan (09:00 CET): Review spec status, identify next spec
2. Build cycle (09:30-12:00): TDD implementation
3. Integration check (14:00): Full test suite
4. Evening report (21:00 CET): Push to repo, update results.tsv

### Phase 2: Autoresearch Loop (Post-Milestone 1)
**CS = 0.40 x PA + 0.30 x US + 0.30 x SE**

### Keep/Discard Rules
**Auto-evolve:** Prompts, data normalization, UI, tests, performance
**Requires approval:** Biomarker tiers, Fitness Equation, crypto protocols, wearable integrations, CC layer architecture
